In the Solution Explorer, right click the catool solution, select Rebuild Solution, it will build the solution and output the generated files to the binary directory of the catool console application. In your command line, go to that binary directory and now we can start to play with the catool.

  • Calculates 128 + 255
  • Calculates 12 * 35
  • Calculates 35 - 23, and do not print the banner information
  • Get the version information
  • When providing no argument, the help screen is displayed

Now you can see how easy it is to build a console application which has complex syntaxes. You don't need to parse the arguments and values, you don't need to maintain the banner information, you don't need to write additional code to tell the user how to use your application, and you don't need to write the syntax definition by yourself.

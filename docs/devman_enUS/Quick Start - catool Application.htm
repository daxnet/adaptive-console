In this chapter, we will introduce how to use ACF to develop a console application which performs arithmetic operations on two given integers. For convenience, we give the name catool (calculation tool) to this application and in the following chapters, we will discuss the design concepts of ACF by taking catool as examples. 

There are four kinds of arithmetic operations on two integers: addition, subtraction, multiplication and division. So in our console application, we should specify the kind of the operation that we are going to take. Also, we must provide the two integer numbers in the command line.

For the users who don't want to see verbose information when the program performs calculation, we will provide a switch called "nologo", which will suppress the display of the application banner information.

We will also provide a syntax with which users can get the version information about the running application.

In addition, when user provides invalid argument, the help screen must be displayed.

From the requirement listed above, we can now figure out the syntaxes that needs to be supported by catool application.

1. catool.exe /m|/method:add|sub|mul|div num1 num2 [/nologo]
2. catool.exe /version|/ver

The first syntax has four arguments, the first argument is for specifying the calculation method (addition, subtraction, multiplication or division), the second and the third arguments are for specifying the numbers on which the arithmetic operation will be applied. The last argument is a switch, which is used for determining whether the banner information should be suppressed.

The second syntax has only one argument: the /version or /ver. By specifying this argument, the application will print the version information on the screen.

Don't worry about the requirement which needs help information be displayed when the invalid arguments are provided. This is totally handled by ACF.

Now let's start and see how the catool console application is implemented by using ACF.

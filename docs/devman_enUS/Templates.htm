Developers can customize the help screen generated by ACF. In order to customize the banner and help body structure of the help screen, please override the BannerTemplate and HelpBodyTemplate abstract properties in your provider class. In order to customize the syntax line and the description for the option contract, please override the Syntax and HelpText abstract properties in the contract class.

The help screen consist of two parts: Banner and HelpBody. The Banner is mainly for displaying the brief information of the application. By default, the Banner shows the following elements:

  • Application Logo (Returned by the overrided Logo property in your provider class)
  • Copyright Information (Defined on the AssemblyCopyright property of the entry assembly)
  • Application Description (Returned by the override Description property in your provider class)

And the HelpBody shows the followings:

  • Syntax line for each option contract (syntax)
  • Syntax description for each syntax line

You can use templates in the BannerTemplate and HelpBodyTemplate properties to define your customized help screen. You cannot use templates in the Syntax and HelpText properties, however, the Attribute and OptionAttributes protected member in your contract class may help you to get the required information while you are going to generate your customized syntax lines and syntax description.

For more information about templates, please refer to the Template Reference.

Some console application may provide the syntax that just carries only one option. For example, gacutil.exe provides a syntax that just have a /? option, which may provide detailed help information on the program. Actually there are many console applications that will have such kind of syntax defined, some application defines the syntax for providing the help screen, some for displaying the system status and others may use it to print the version number of the application. No matter how the application behaves, we call such kind of syntax an Exact contract in ACF.

Take catool.exe as an example, it has the following syntaxes (or option contracts) that are of the type of Exact.

catool.exe /version|/ver|/v
catool.exe /help|/h|/v

The first one is for displaying the version information of the application, and the second is for displaying the help screen.

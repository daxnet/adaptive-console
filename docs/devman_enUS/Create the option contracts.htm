In ACF, each option contract represents a command line syntax for console application, so create and define the option contracts is the most important and common task while using ACF to build your applications. In this chapter, we are going to define two different contracts for catool, the first is to display the version information, and the other is to perform the arithmetic operations on the given integers.

Currently you just need to follow the steps that we have demonstrated. Don't ask why, the answers to the questions that confused you can be found under the Design of the Adaptive Console Framework node of this developer's manual.

What's Adaptive Console Framework?

Adaptive Console Framework (ACF) is the console application development framework that makes it possible to:

  • Simplify the command line argument parsing and validation
  • Support multiple command line syntax
  • Generate command line syntax and argument description automatically
  • Support alias names for command line arguments
  • Parse and recognize command line arguments orderlessly
  • Customize the help screen by using templates
  • Support Test Driven Design (TDD). ACF makes your console application testable
  • Get flexibility of the console application. You can add or remove command line syntax just by using application configuration

If you have one or more requirement listed above, ACF is for you.

System Requirements

Your machine must meet the following requirements if you want to work with ACF:

  • Operating System: Windows XP, Windows Server 2003 or Windows Vista (Not tested)
  • Microsoft .NET Framework 3.5 or higher

ACF supports the following templates while you are overriding the BannerTemplate and HelpBodyTemplate methods in your console application provider.

Template

Description

Remarks

<%app_asm_ver%> The value of the AssemblyVersion property in the entry assembly attribute.  *
<%app_asm_title%> The value of the AssemblyTitle property in the entry assembly attribute.  *
<%app_asm_desc%> The value of the AssemblyDescription property in the entry assembly attribute.  *
<%app_asm_product%> The value of the AssemblyProduct property in the entry assembly attribute.  *
<%app_asm_copyright%> The value of the AssemblyCopyright property in the entry assembly attribute.  *
<%app_asm_company%> The value of the AssemblyCompany property in the entry assembly attribute.  *
<%app_logo%> The logo string of the console application. This is the string returned by the Logo abstract property of ConsoleApplicationBase which must be overrided by the application provider.
<%app_desc%> The description string of the console application This is the string returned by the Description abstract property of ConsoleApplicationBase which must be overrided by the application provider.
<%app_syn_lines%> The console application syntax lines that are generated automatically by ACF.

For example, in catool application, it refers to the following lines in the help screen:

catool.exe </method|/m:> [/nologo] p1 p2
catool.exe /version|/ver

<%app_syn_descs%> The detailed syntax description for each syntax line and each argument in the syntax.

For example, in catool application, it refers to the following lines in the help screen:

> Performs arithmetic calculation
  /method|/m:value (required):
        Specifies the calculation method
                add: Performs an addition calculation
                sub: Performs a subtraction calculation
                mul: Performs a multiplication calculation
                div: Performs a division calculation

  [/nologo]:
        Suppress the display of banner information

> Display the version information
  /version|/ver:
        Display the version information

* These templates refer to the assembly properties set on the entry assembly (Commonly the .exe console application assembly):

ACF will configure your console application by using application configuration file, so firstly you should add a App.config file to your console application.

  1. Right click on the catool application in Solution Explorer, select Add | New Item
  2. In the Add New Item dialog, select Application Configuration File and click Add
  3. Modify the App.config file like the following:

        1 <?xml version="1.0" encoding="utf-8" ?>

        2 <configuration>

        3   <configSections>

        4     <section name="AdaptiveConsole"

        5             type="AdaptiveConsole.Config.AdaptiveConsoleConfigHandler, AdaptiveConsole"/>

        6   </configSections>

        7   <AdaptiveConsole provider="Catool.Provider.CatoolApplication, Catool.Provider">

        8     <repositories>

        9       <repository name="contracts" assembly="Catool.Contracts"/>

       10     </repositories>

       11   </AdaptiveConsole>

       12 </configuration>

A ValueList option type refers to the options that have a list of option values.

It is also widely used in console applications. Suppose there is a console application that takes multiple files as input files, we can define a /input option which is the ValueList type, and use this application like this:

D:\>my_app /input:a.dat,b.dat,c.dat ...

The separator character, which can be customized within ACF, must be used between the values in order to separate the values in a ValueList option.

A SingleValue option type refers to the options that have only one option value.

For example, in the C# compiler options, there is an option called /target, it has four different values that can be used: exe, winexe, library and module. When use this option to specify the output target, we must use /target:xxx, where xxx stands for exe, winexe, library or module. So, the /target option will have a single value defined along with it.

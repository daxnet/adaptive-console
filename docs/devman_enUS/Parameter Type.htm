All the arguments that is not the option should be considered as a parameter.

Please refer to the following catool syntax:

D:\>catool.exe /method:add 20 30

Here 20 and 30 are the parameters.

It is very easy to tell which arguments are options and which are parameters in ACF. You can use the LINQ below to get all the parameter arguments in a command line:

var parameters =

        from arg in args

        where arg.Category == ArgumentCategory.Parameter

        select arg.Argument;

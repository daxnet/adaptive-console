A Switch option type refers to the options that represent the meaning of Yes or No.

Switch options are also frequently used in console applications. For example, some application provides the /nologo option which is used for suppressing the display of logo information. You can either specify or not specify this option. If you have specified this option in the command line, that means you wish to turn on this switch and suppress the display of logo information, otherwise the logo information is displayed when it is executing.
